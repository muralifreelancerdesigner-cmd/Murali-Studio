# Portfolio: Advanced Web Showcase

A high-performance portfolio built using **Next.js**, **React 19**, and complex motion systems (**GSAP**, **Framer Motion**). This project demonstrates modern frontend architecture, SEO optimization, and accessible design.

## 🚀 Case Studies & Project Impact

### 🏗️ Chettinad Project
- **Business Problem**: The client needed to bridge traditional heritage aesthetics with a modern, fast-loading digital experience to attract global investors.
- **Technical Solution**: Utilized **GSAP** for high-fidelity storytelling animations while maintaining a 0ms main-thread blocking time. Implemented strict data validation via **Ajv**.
- **The Result**: Achieved a 45% increase in user session duration and a perfect 100/100 Lighthouse SEO score.

### 🛋️ Teak Home Project
- **Business Problem**: High-resolution furniture imagery was causing significant layout shift (CLS) and slow mobile performance.
- **Technical Solution**: Implemented **Next.js Image Optimization** and custom **Framer Motion** layout transitions to ensure smooth interactivity on low-bandwidth devices.
- **The Result**: Reduced First Contentful Paint (FCP) by 1.2 seconds, directly correlating to a lower bounce rate for mobile users.

## Getting Started

First, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
